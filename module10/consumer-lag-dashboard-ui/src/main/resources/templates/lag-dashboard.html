Important Clarification

Consumer lag is a real Kafka concept. It is based on offsets.

Kafka lag means the difference between the latest offset in the topic and the committed offset of the consumer group.

The numbers shown in this UI are a simple application-level approximation: produced count minus consumed count.

For real Kafka lag, use the command: kafka-consumer-groups --describe --group slow-reporting-group
