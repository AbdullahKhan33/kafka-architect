UI-7: Delivery Status Tracking using Multiple Kafka Events

Kafka Concept

This UI publishes multiple delivery events for the same order into delivery-status-topic. 

ORDER_PLACED
    ↓
ORDER_PACKED
    ↓
ORDER_SHIPPED
    ↓
OUT_FOR_DELIVERY
    ↓
DELIVERED

Every status change is a separate Kafka event. The dashboard consumer reads the events and updates the latest status.
